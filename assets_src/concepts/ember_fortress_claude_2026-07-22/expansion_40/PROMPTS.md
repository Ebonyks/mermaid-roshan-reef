# Ember Fortress expansion-40 prompt provenance

Generated with the OpenAI built-in image-generation tool on 2026-07-22. The
existing `contact_sheet.png` was supplied only as a style reference. Each of
the forty named roles used a separate generation call and produced one square
modeling card; no card was sliced from a combined sheet.

## Shared prompt contract

Use case `stylized-concept`; individual 2D Blender modeling reference card;
exactly one named object or explicitly named cluster; original generic volcanic
fantasy; dense spherical adventure-world richness only as an abstract goal;
no reproduction of Nintendo, Mario Galaxy, Bowser, Zelda, or other copyrighted
designs. Match the Ember reference's cel-shaded pastel toy-diorama, rounded
low-poly construction, navy-purple outlines, charcoal/plum basalt, lavender or
aqua shadows, and sparse coral-orange emission. Present an isolated object on
a cream modeling background with large three-quarter plus front/side views and
an unlabeled meter ruler. No extra scene, character, face, eyes, mouth, logo,
text, watermark, franchise symbol, thin spike, weapon, or decorative clutter.

## Role-specific prompt set

The exact role descriptions and scale requirements are recorded row-by-row in
`CLAUDE_EXPANSION_40_MANIFEST.csv`. They cover ten architecture modules, ten
terrain pieces, ten interactive/dressing props, and ten ambient/guidance roles.
The CSV scale and child/state contracts are authoritative over any approximate
markings produced inside the generated cards.
